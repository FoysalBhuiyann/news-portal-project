1.Difference between var, let and const?

Ans:var:- It can be declared without initialization.
let:- It can be declared without initialization.
const:- It cannot be declared without initialization.

2.The Difference Between Regular Functions and Arrow Functions?

Ans:Arrow Function:- Arrow functions are a new way to write anonymous function expressions, and are similar to lambda functions in some other programming languages, such as Python. Arrow functions differ from traditional functions in a number of ways, including the way their scope is determined and how their syntax is expressed

Regular Function:- In mathematics, a regular function in the sense of algebraic geometry is an everywhere-defined, polynomial function on an algebraic variety V with values in the field K over which V is defined.
